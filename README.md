# 📚 Exercícios de HTML e CSS

Este repositório contém uma coleção de exercícios práticos de **HTML** e **CSS** para quem está aprendendo **front-end sozinho**. O objetivo é criar um espaço de prática constante, reforçando conceitos fundamentais e ajudando a construir projetos cada vez mais complexos.

---

## 🎯 Objetivo

- Aprender a estruturar páginas web com HTML.
- Estilizar páginas com CSS.
- Praticar layouts responsivos e modernos.
- Construir pequenos projetos que simulam situações reais.
- Ganhar confiança para criar seus próprios sites do zero.

---

## 🗂 Estrutura do repositório

O repositório está organizado por exercícios, cada um com uma pasta própria contendo os arquivos HTML e CSS:

### Exercícios

| Exercício | Descrição |
|-----------|-----------|
| **exe01** | Menu fixo e conteúdo semelhante a um blog.<br>Arquivos: `index.html` e `style.css` |
| **exe02** | Munu lateral - Sidebar.<br>Arquivos: `index.html` e `style.css` |
| **exe03** | [Rodapé (footer) simples ].<br>Arquivos: `index.html` e `style.css` |


> Dica: cada pasta é independente. Você pode abrir o `index.html` diretamente no navegador e testar o exercício.
