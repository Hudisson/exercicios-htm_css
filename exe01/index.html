Descrição do Projeto

A seguir está uma breve descrição deste exercício

Este projeto teve como objetivo criar um menu de navegação fixo no topo da página, utilizando apenas HTML e CSS, além de estruturar o conteúdo de forma centralizada e com boa legibilidade, seguindo um estilo semelhante ao de blogs.

Inicialmente, foi construída a estrutura HTML com a tag <nav>, responsável por representar o menu de navegação. Dentro dela, foram adicionados a logo do site e uma lista de links (<ul> e <li>), que funcionam como as opções do menu. Abaixo do menu, foi criada uma área de conteúdo principal para exibir textos da página.

No CSS, o primeiro passo foi aplicar um reset básico para remover margens e espaçamentos padrão dos navegadores, garantindo consistência visual. Em seguida, o menu foi configurado com position: fixed e top: 0, fazendo com que ele permaneça sempre visível no topo da tela, mesmo durante a rolagem da página.

Para organizar os elementos do menu (logo e links), foi utilizado o Flexbox (display: flex), permitindo alinhar os itens horizontalmente e distribuir o espaço entre eles de forma equilibrada. Também foram aplicados estilos visuais, como cores, espaçamentos internos (padding) e efeitos de hover, melhorando a experiência do usuário.

Outro ponto importante foi o ajuste do conteúdo da página. Como o menu é fixo, foi necessário adicionar uma margem superior (margin-top) na área de conteúdo para evitar que ele fique escondido atrás do menu.

Além disso, foi criado um container centralizado com largura máxima (max-width) e margens automáticas (margin: 0 auto), deixando o layout mais organizado e semelhante ao de um blog. Foram adicionados também espaçamentos entre parágrafos (margin-bottom) e ajustes de tipografia, como altura de linha (line-height), para melhorar a leitura.

Por fim, foi implementada a indentação na primeira linha dos parágrafos com text-indent, proporcionando um estilo mais tradicional de texto, semelhante a livros.

O resultado final é uma página com um menu fixo funcional, layout centralizado e conteúdo bem estruturado, oferecendo uma navegação clara e uma leitura confortável.
