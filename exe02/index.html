Conteúdo

Foi criado um menu lateral utilizando HTML e CSS com o objetivo de organizar a navegação do site. A estrutura HTML foi construída com uma barra lateral (sidebar) contendo uma lista de links, e uma área principal para o conteúdo.

No CSS, utilizamos Flexbox para posicionar o menu ao lado do conteúdo. A barra lateral recebeu uma largura fixa, altura total da tela e uma cor de fundo escura para destacar a navegação.

Os itens do menu foram estilizados removendo os marcadores da lista e ajustando espaçamentos. Cada link foi configurado com display: block para ocupar toda a área do item, permitindo aplicar efeitos visuais mais completos.

Também foi adicionado um efeito de hover, que altera o fundo do item quando o usuário passa o mouse, melhorando a experiência de uso. Para deixar a transição mais suave, utilizamos a propriedade transition no CSS.

Com isso, o resultado final é um menu lateral funcional, organizado e com interação visual agradável.
